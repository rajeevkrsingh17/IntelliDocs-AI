# Design Document

## Project Name

IntelliDocs-AI

## Problem Statement

Segment 3 – Foundations of Applied Machine Learning

Problem Code: I2 – Document Q&A (RAG over a Focused Corpus)

## Objective

To build an AI-powered document question answering system that allows users to upload documents and receive relevant answers with citations using Retrieval-Augmented Generation (RAG).

## Target Users

- Students
- Researchers
- Professionals

## Expected Outcome

Users will be able to ask questions from uploaded documents and receive accurate, context-aware responses.
