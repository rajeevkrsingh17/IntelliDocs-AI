# Technology Stack

## Overview

The IntelliDocs-AI project uses Python-based technologies to build the foundation of an AI-powered document question answering system. The selected tools are chosen for simplicity, scalability, and compatibility with Retrieval-Augmented Generation (RAG).

| Component | Choice | Why This Choice |
|-----------|--------|-----------------|
| Programming Language | Python | Widely used for AI, machine learning, and rapid application development. |
| PDF Processing | PyMuPDF | Efficient and reliable library for extracting text from PDF documents. |
| AI Framework *(Planned)* | LangChain | Simplifies building Retrieval-Augmented Generation (RAG) pipelines. |
| Vector Database *(Planned)* | ChromaDB | Stores embeddings and enables semantic document retrieval. |
| User Interface *(Planned)* | Streamlit | Quickly builds interactive web applications with Python. |
| Version Control | Git & GitHub | Tracks project history and supports collaborative development. |
